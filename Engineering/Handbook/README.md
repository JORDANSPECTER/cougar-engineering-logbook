# Cougar Engineering Handbook

Welcome to the Cougar Engineering Handbook.

This handbook defines the permanent engineering standards used throughout Cougar Technologies.

Unlike the Engineering Logbook, which records the day-to-day history of development, the Engineering Handbook defines **how** engineering is performed.

These documents are intended to remain relatively stable while the platform evolves.

---

# Purpose

The Engineering Handbook exists to ensure that Cougar is built with consistency, quality, and long-term maintainability.

Every engineer contributing to Cougar should follow the standards defined in this handbook.
