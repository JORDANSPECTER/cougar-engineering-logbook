# Engineering Principle

A milestone represents a significant engineering achievement that advances the Cougar platform.

Every milestone should:

- Solve a meaningful engineering problem.
- Improve the platform's architecture or capabilities.
- Be documented through the Engineering Logbook.
- Be associated with a Cougar OS release.
- Include supporting Architecture Decision Records (ADRs) when appropriate.
